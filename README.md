# Teslo API
1. Clonar el proyecto.
2. Ejecutar ```pnpm i```.
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```.
4. Cambiar las variables de entorno.

5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar proyecto: ```pnpm start:dev```
7. Ejecutar Seed
```
http://localhost:4001/api/seed
```
